# Wedding Website

A beautiful, responsive wedding website built with HTML, CSS, and JavaScript.

## Features

- ✦ Elegant blue and purple theme with traditional Indian wedding elements
- 🎭 Multi-page site with 8 different sections
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🎨 Professional styling with mandala decorative elements
- 📸 Photo gallery section
- 💌 RSVP with Google Forms integration
- 📝 Guest Book with email submissions
- ❓ FAQ/Q&A section
- 📧 Contact form
- ⚡ Fast, lightweight static site

## Pages Included

1. **Home** - Welcome page with event overview
2. **Our Story** - Love story and engagement details
3. **Wedding Details** - Event schedules, venue information, dress code
4. **Photo Gallery** - Image gallery with lightbox feature
5. **RSVP** - Embedded Google Form for RSVPs
6. **Guest Book** - Leave wishes and blessings
7. **Contact** - Contact information and contact form
8. **Q&A** - FAQ section with expandable questions

## Setup Instructions

1. Clone or download this repository
2. Upload your logo to `assets/images/logo` (as any image format)
3. Upload your photos to `assets/images/` directory
4. Customize the content in each HTML file with your information
5. Update contact details, email addresses, and links as needed

## Customization

### Logo & Images
- Replace `assets/images/logo` with your wedding logo
- Add photos to `assets/images/` and reference them in the pages

### Google Form Integration
- Update the Google Form embed link in `rsvp.html`
- The form should point to your Google Sheet

### Contact Information
- Update contact details in `contact.html`
- Update email addresses throughout the site

### Theme Colors
- Modify CSS variables in `assets/css/style.css`:
  - `--primary-blue: #2c3e7f`
  - `--primary-purple: #663399`
  - `--accent-gold: #d4af37`

## Hosting on GitHub Pages

1. Create a GitHub account if you don't have one
2. Create a new repository named `yourusername.github.io` (or any name)
3. Upload all files to the repository
4. Enable GitHub Pages in repository settings
5. Your site will be live at `https://yourusername.github.io/`
